# Proyección y Perspectiva

En el siguiente proyecto se crearon tres escenas:
La primera consiste en un entorno en el cual los objetos cambian de tamaño según la distancia a la que se colocaron del jugador después de interactuar.
La segunda escena consiste en un plataformero 3D con elementos 2D (monedas), con los cuales se puede interactuar mediante colisiones 3D en general.
En la tercera escena se implementan los cambios de perspectiva en la cámara para poder ubicarse mejor en un laberinto.

## Controles 3D
- WASD - moverse
- Click izquierdo del ratón - Interactuar con objetos
- F - Cambiar perspectiva de cámara
- Esc - salir

## Controles 2D
- A - Izquierda
- D - Derecha
- Barra espaciadora - saltar
- Esc - salir
